# Stór og Lítill 🔷

Einfalt offline kennsluforrit fyrir börn sem kennir muninn á **stór** og **lítill** með formum og hljóði.

## Hvernig á að opna

1. Klónaðu eða sæktu þessa geymslu.
2. Tvísmelltu á **`index.html`** eða dragðu hana yfir í vafrann.
3. Engin netsamband eða þjónn þarf — forritið virkar beint úr skrám (`file://`).

## Viðeigandi vafrar

| Vafri | Virkar? | Íslenskt TTS |
|-------|---------|--------------|
| Chrome / Edge (Windows, Android) | ✅ | Líklega (Windows TTS stillingar) |
| Safari (macOS / iOS) | ✅ | Til á macOS/iOS með íslenskri rödd |
| Firefox | ✅ | Sjaldnar íslenskt |

## TTS-athugasemdir

Forritið notar **Web Speech API** (SpeechSynthesis) til að segja „stór" eða „lítill" á íslensku.

- Ef íslensk rödd er til á tækinu verður hún notuð sjálfkrafa.
- Ef ekki: önnur rödd er notuð en textinn er íslenskur.
- **Windows:** Stillingar → Tími og tungumál → Talgreining → bættu við íslensku.
- **macOS/iOS:** Kerfisstillingar → Accessibility → Spoken Content → System Voice → Icelandic.
- Ef hljóð heyrist ekki skaltu ganga úr skugga um að tækið sé ekki á þögulmóð og smella á **🔊 Spila aftur**.

## Leikurinn

- **20 spurningar** í hverri lotu.
- Hljóð segir „stór" eða „lítill" — smelltu á rétta formið!
- Rétt svar → grænn litur; rangt svar → rauður litur.
- Framvinda og stig sjást efst á skjá.
- **🔊 Spila aftur** endurtekur hljóðið.

## Form í leiknum

Hringur, ferhyrningur, þríhyrningur, stjarna, demant, hjarta, sexhyrningur, plús.
